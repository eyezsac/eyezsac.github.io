Let's make some word clouds from my thesis! 

We're going to use R to read in the data, tidy it up, and then spit out some nice looking word clouds (hopefully). This is a good technique if you're wanting to make some easy visualisations for presentations, or just want to shove a quick elevator pitch for your work on a page.

``` r
# install packages if you don't have them already
install.packages("tidyverse") # duh
install.packages("ggwordcloud") # for the word cloud in ggplot
install.packages("tidytext") # for text tidying and mining

# load packages
library(tidyverse)
library(ggwordcloud)
library(tidytext)
```

Next up, we'll need to import the text data from my thesis. I've already converted it into a .txt file, so we can read it in using the `readLines()` function.

``` r
thesis_text <- readLines("assets/data/wordcloud/thesis.txt")
head(thesis_text) # check it's read in properly
```

Now we've got the data, we need to format in a way that makes sense. ggwordcloud needs a data frame with two columns: one for the words, and another for the frequency of those words. Here we're going to use the tidytext package to turn the text into a data frame, remove empty lines, and then count the frequency of each word.

We're also going to remove punctuation, and filter out words less than 3 characters long, as these usually aren't informative. Don't want to fill it up with "the", "and", "is", "we", etc.

``` r
# let's pre-process the data
thesis_df <- thesis_text |> 
  tibble() |> # turn it into a data frame
  rowid_to_column() |> # add row.ids
  filter(!thesis_text == "") # remove empty lines

# now we need to tokenise + count to get a frequency table of words! 
word_freq <- thesis_df |>
  # remove anything in brackets and special characters
  # removing anything in brackets because it's usually a reference
  mutate(thesis_text = str_replace_all(thesis_text, "\\([^)]*\\)", ""),
         thesis_text = str_replace_all(thesis_text, "[^A-Za-z\\s]", "")) |> 
  # tokenise your words
  unnest_tokens(word, thesis_text) |>
  # count!!
  count(word, sort = TRUE) |> 
  # remove some really low-length strings
  filter(str_length(word) >= 3) 

# prepare a string of words to convert to uppercase
needs_upper <- c("rna", "dna", "deg", "dmr", "chg", "chh")

# let's do some final filtering! 
word_freq <- word_freq |>
  # remove some common words 
  filter(!word %in% c("the", "and", "for", "that", "with", "are", 
                      "this", "was", "from", "which", "have", "but", 
                      "not", "all", "were", "can", "due", "its", 
                      "each", "has", "these", "like")) |> 
  # fix capitalisation.
  # some edge cases will require manual editing
  mutate(word = case_when(word %in% needs_upper ~ toupper(word), 
                          word == "dmrs" ~ "DMRs", 
                          word == "degs" ~ "DEGs", 
                          TRUE ~ word)) |> # essential
  filter(n > 30) # filter out words that don't appear much 
```

Now we can get to making a word cloud! We're going to use ggwordcloud for this. It's using normal ggplot2 syntax so we can make some really easy adjustments. This might take a second for it to render...

``` r
word_freq |>
  head(20) |>
  ggplot(aes(label = word, size = n)) +
  geom_text_wordcloud() +
  scale_size_area(max_size = 15) + # adjust the max size of the words
  theme_minimal() 
```

It worked! Now we can adjust some of the aesthetics and give it some colour and direction. Here we're going to grab the top 100 words, randomly assign them an angle, and then randomnly assign them a colour. Adjust these to your heart's content :)

``` r

word_freq |>
  slice_max(n, n = 100) |> # grab the top 100 words
  mutate(angle = 45 * sample(c(1,0), n(), replace = TRUE, prob = c(2, 3))) |> 
  mutate(colour = factor(sample.int(6, n(), replace = TRUE))) |>
  ggplot(aes(label = word, size = n, angle = angle, colour = colour)) +
  scale_colour_manual(values = c("#52B69A","#34A0A4","#168AAD","#1A759F",
                                 "#1E6091","#184E77")) + 
  geom_text_wordcloud_area(shape = "square") +
  scale_size_area(max_size = 50, trans = power_trans(1/.7)) +
  theme_minimal() -> wordcloud_gg

# now let's save the file with ggsave! 
wordcloud_gg |> 
ggsave(filename = "assets/images/wordcloud/wordcloud_thesis.png", 
       width = 6, height = 4, dpi = 300, device = "png", bg = "transparent")
```

Aaaaand here we go: a wordcloud of my thesis. We can do some further tweaks, but I'm pretty happy with this one.
